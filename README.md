# 🌦️ Weather Detection App

A simple and interactive weather application that displays real-time weather data using the OpenWeather API.

## 🚀 Features
- Search weather by city
- Animated moving clouds using CSS
- Displays temperature, humidity, and wind speed
- Responsive and user-friendly design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
- Git & GitHub
